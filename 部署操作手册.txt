一、电脑端运行（H5浏览器版）
================================================================================

第1步：安装 Node.js
    下载地址：https://nodejs.org （选择 LTS 版本）
    安装完成后打开 cmd，输入 node -v 确认版本号

第2步：初始化数据库
    1. 打开 MySQL Workbench，连接到本地 MySQL（root账号）
    2. 菜单 File → Open SQL Script → 选择 server/database/init.sql
    3. 点击闪电图标执行全部 SQL
    4. 左侧 SCHEMAS 出现 health_management 数据库即成功

第3步：配置后端
    打开 server/.env 文件，确认以下配置：
    DB_HOST=localhost
    DB_PORT=3306
    DB_USER=root
    DB_PASSWORD=你的MySQL密码
    DB_NAME=health_management

第4步：启动后端
    打开 cmd 窗口，输入：
    cd server
    npm install
    npm start
    看到"服务器运行在端口 3000"和"数据库连接成功"即可。
    注意：这个窗口不要关闭！

第5步：启动前端（推荐用 HBuilderX）
    1. 打开 HBuilderX（桌面上有图标）
    2. 文件 → 打开目录 → 选择 client 文件夹
    3. 运行 → 运行到浏览器 → Chrome
    4. 浏览器自动打开系统页面


二、手机 App 打包注意事项
================================================================================

（1）修改后端地址（最重要！）
----------------------------------------------------------------------
    打开 client/utils/request.js，修改前两行：

    把 http://localhost:3000/api 改为 http://你电脑的IP:3000/api
    把 http://localhost:3000 改为 http://你电脑的IP:3000

    查看电脑IP方法：
    打开 cmd → 输入 ipconfig → 找到"IPv4 地址"（如 192.168.1.100）

    重要：不能用 localhost！手机无法访问电脑的 localhost！
    手机和电脑必须连接同一个 WiFi！


（2）修复图标不显示问题
----------------------------------------------------------------------
    原因：底部导航栏图标使用的是 SVG 格式，但 UniApp 原生 App 和
         微信小程序不支持 SVG 图标，只支持 PNG 格式。

    解决方法：
    步骤一：将 client/static/tabbar/ 目录下的 8 个 SVG 文件转换为 PNG
            可以用在线工具：https://svgtopng.com
            建议图片尺寸：81 x 81 像素
            转换清单：
            home.svg        → home.png
            home-active.svg → home-active.png
            health.svg        → health.png
            health-active.svg → health-active.png
            hospital.svg        → hospital.png
            hospital-active.svg → hospital-active.png
            mine.svg        → mine.png
            mine-active.svg → mine-active.png

    步骤二：打开 client/pages.json 文件，找到 tabBar 部分
            把所有 .svg 后缀改成 .png：
            "iconPath": "static/tabbar/home.svg"
            改为
            "iconPath": "static/tabbar/home.png"

            一共需要改 8 处（4个 iconPath + 4个 selectedIconPath）


（3）修复定位失败问题
----------------------------------------------------------------------
    原因分析：
    - App端：需要开启手机GPS并授权定位权限
    - H5端：浏览器定位需要 HTTPS（HTTP下会被禁止）
    - 微信小程序：需要配置定位权限声明

    解决方法：

    App端：
    - 确保手机设置中打开了GPS定位
    - App首次使用时弹出的定位权限请求要点"允许"
    - 在 manifest.json 中确认包含以下权限：
      android.permission.ACCESS_FINE_LOCATION
      android.permission.ACCESS_COARSE_LOCATION

    H5端：
    - 必须用 HTTPS 访问才能使用浏览器定位
    - HTTP 下系统会自动降级为 IP 定位（精度较低）

    高德地图Key配置：
    - 确认 server/.env 中的 AMAP_KEY 已配置有效的高德地图 API Key
    - 申请地址：https://lbs.amap.com
    - 注册登录 → 控制台 → 应用管理 → 创建应用 → 添加 Key
    - 选择"Web服务"类型的 Key


（4）HBuilderX 云打包生成 APK
----------------------------------------------------------------------
    1. 用 HBuilderX 打开 client 目录
    2. 双击 manifest.json → 基础配置 → 点击"重新获取" AppID
       （需要登录 DCloud 账号）
    3. 菜单：发行 → 原生App-云打包
    4. 配置：
       Android 包名：com.health.family
       证书选择：使用公共测试证书
    5. 点击"打包"，等待 5-15 分钟
    6. 完成后控制台会显示下载链接
    7. 下载 APK 传到手机上安装即可


三、微信小程序发行操作
================================================================================

前置条件：
    - 注册微信小程序账号：https://mp.weixin.qq.com
    - 获取小程序 AppID（在微信公众平台 → 开发 → 开发设置中查看）
    - 后端必须部署到有域名和SSL证书的服务器上（微信要求HTTPS）

（1）配置 AppID
    打开 client/manifest.json
    在"微信小程序配置"部分填入你的 AppID

（2）修改后端地址
    微信小程序要求后端必须使用 HTTPS（不支持 HTTP！）
    后端需要部署到有域名和 SSL 证书的服务器上
    修改 client/utils/request.js 中的地址为：
    https://你的域名/api

（3）配置合法域名
    登录微信公众平台 → 开发 → 开发设置 → 服务器域名：
    request 合法域名：https://你的域名
    uploadFile 合法域名：https://你的域名

    开发阶段可在微信开发者工具中勾选"不校验合法域名"跳过此限制。

（4）修复图标不显示
    同 App 端，将 SVG 图标转换为 PNG 格式，修改 pages.json 中的路径。

（5）修复定位失败
    打开 client/manifest.json，在微信小程序配置中添加定位权限声明：
    "mp-weixin" 部分添加 permission → scope.userLocation → desc
    描述填写："您的位置信息将用于查找附近医院"

（6）构建并上传
    1. HBuilderX → 发行 → 小程序-微信
    2. 打开微信开发者工具
    3. 导入项目：client/dist/dev/mp-weixin 目录
    4. 点击"上传"提交到微信后台
    5. 登录微信公众平台 → 版本管理 → 提交审核


四、常见问题排查
================================================================================

问题：后端启动报"数据库连接失败"
原因：MySQL未启动或密码错误
解决：启动MySQL服务，检查 server/.env 中的密码是否正确

问题：页面空白或接口报错
原因：后端未启动或地址配错
解决：确认后端正在运行，检查 client/utils/request.js 中的地址

问题：App底部图标不显示
原因：SVG格式不被App和小程序支持
解决：转为PNG格式，修改 pages.json 中的图标路径

问题：定位失败
原因：权限未授权 或 使用HTTP不支持定位
解决：开启GPS、授权定位权限、使用HTTPS访问

问题：微信小程序请求失败
原因：域名未在微信后台配置
解决：在微信公众平台配置合法域名，或开发时跳过校验

问题：头像和图片不显示
原因：SERVER_URL 配置错误
解决：检查 client/utils/request.js 中 SERVER_URL 是否正确

问题：AI健康建议功能不显示
原因：DeepSeek API Key 未配置
解决：在 server/.env 中配置 DEEPSEEK_API_KEY
      申请地址：https://platform.deepseek.com （新用户有免费额度）
