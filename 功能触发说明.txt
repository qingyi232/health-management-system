================================================================================
          家庭健康管理系统 — 功能触发操作说明
================================================================================


一、操作前提（必须先完成）
================================================================================

第1步：注册并登录账号

第2步：添加家庭成员
    首页 → 家庭成员区域 → 点击"+"添加
    ⚠️ 出生日期必须填写（个性化推荐靠出生日期算年龄来匹配文章分类）

第3步：在首页点击选中该成员
    成员头像下方出现高亮状态表示已选中


二、触发"数据分析 + 趋势图"
================================================================================

第4步：先录入健康数据（至少录几条）
    首页 → 健康概览区域 → 点击任意指标（如"体重""血压"）进入添加页面

    建议录入操作：
    - 录 2条以上体重数据（不同日期）→ 统计页才会显示"体重变化"
    - 录 几条血压数据 → 统计页才会显示"平均血压"
    - 录 几条血糖数据 → 统计页才会显示"平均血糖"
    - 录 步数数据 → 统计页才会显示"累计步数"

第5步：进入数据分析页面
    底部导航"健康" → 点击"数据分析"
    - 顶部确认当前成员是否正确
    - 健康统计卡片会显示近30天的汇总
    - 趋势图区域可以切换"体重/血压/血糖/心率"和"7天/30天/3月"

    ⚠️ 如果趋势图显示"暂无数据"：说明在所选的时间范围内没有对应类型的
    记录。比如选了"7天"但数据是30天前录的，就看不到。


三、触发"个性化推荐"
================================================================================

第6步：回到首页查看
    首页底部"个性化推荐"板块
    系统会根据当前选中成员的 年龄 + 健康数据 自动匹配文章：

    ┌─────────────┬──────────────────────────────────┐
    │ 成员年龄     │ 推荐的文章类别                    │
    ├─────────────┼──────────────────────────────────┤
    │ 18岁以下     │ 青少年健康                        │
    │ 18-40岁     │ 运动健身、营养饮食、心理健康        │
    │ 40-60岁     │ 慢病管理、营养饮食                 │
    │ 60岁以上     │ 老年保健、慢病管理、营养饮食        │
    └─────────────┴──────────────────────────────────┘

    额外触发：
    - 如果录入了血压偏高（收缩压>130 或 舒张压>85）的数据，
      会额外推荐高血压相关文章
    - 如果录入了血糖偏高（>6.1）的数据，
      会额外推荐糖尿病相关文章

    每篇推荐文章下面会显示推荐理由标签
    （如"基于您的血压数据""适龄运动建议"等）


四、总结
================================================================================

    这些功能不是"展示型"的，它们依赖用户的真实数据来驱动——
    有数据才有分析，有数据才有个性化推荐。
    请先按上面第1-4步操作，再查看分析和推荐就能看到效果了。
